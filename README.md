# python词云图

利用python进行文本分析并分词(完成作业)



# 安装

在pycham处的Terminal输入以下命令安装相关函数库

```base
pip install -r requriements.txt
```



# 模块化设计

运行demo.py文件即可得到词云图

remove.py   删除文中的非关键字（例如：‘你’，‘我’ ，等字）

word.py   通过jieba进行分词

plat.py   绘制词云图
